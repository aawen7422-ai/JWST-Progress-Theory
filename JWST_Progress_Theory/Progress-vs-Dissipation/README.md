Статья 4: Стремление системы к структурной сложности против диссипативных сил

Суть: Чёрная дыра — не смерть прогресса, а его превращение в гравитацию.

Главная формула
Ξ = (dC/dt) / Γ_diss

· Ξ > 1: сложность побеждает (галактики, планеты)
· Ξ < 1: диссипация побеждает (войды, капля воды)

Новое понятие

Давление прогресса:
P_prog = (dC/dt) × ħ/c² / объём

Это сила, с которой стремление к сложности сопротивляется гравитации.

Что доказано

1. Сверхновая — когда P_prog побеждает гравитацию, прогресс вырывается наружу
2. Чёрная дыра — когда P_prog не может победить, прогресс становится самой гравитацией
3. Граница 3 массы Солнца — это порог, где P_prog перестаёт справляться
