# Аккреционный диск как арена борьбы прогресса: гидродинамическая аналогия и критерий Ξ

**Статус:** Опубликованный препринт
**DOI:** 10.5281/zenodo.18599688 
**Часть:** Исследовательская программа «Теория прогресса как квантовая фаза» (Статья №7)

## Основная идея
Наблюдение за водоворотом в ванной (с его «тенью» и отталкивающими волнами) служит прямой гидродинамической аналогией для аккреционного диска вокруг чёрной дыры. Теория формализует это, вводя **локальный критерий прогресса Ξ_disk(r)**, который разделяет область оттока/ветров (Ξ > 1) и область неотвратимого падения вещества (Ξ < 1).

## Ключевая формула
Локальный критерий структуро­генеза в диске:
```
Ξ_disk(r) = (dC/dt)_pot(r) / (Γ_grav(r) + Γ_visc(r))
```
где `(dC/dt)_pot(r)` — потенциал сложности аккрецирующей оболочки вещества.

## Главный количественный результат
Для чёрной дыры звёздной массы (`M_BH = 10 M_sun`) критический радиус, разделяющий зоны:
```
r_crit ≈ 50 * r_g
```
где `r_g` — гравитационный радиус.
- **При r > r_crit:** Ξ_disk > 1 → **Зона отталкивания** (аналог волн водоворота). Доминируют давление излучения и МГД-силы.
- **При r < r_crit:** Ξ_disk < 1 → **Зона захвата** (аналог тени водоворота). Доминируют гравитация и вязкость.

## Связь с известными феноменами
Теория даёт единую интерпретацию в рамках концепции прогресса для:
*   **Предела Эддингтона** как глобального условия, при котором Ξ_disk приближается к критическому значению.
*   **Формирования релятивистских джетов** как оттока сложности из области, где Ξ_disk > 1.
*   **Ветров аккреционного диска** как проявления «давления прогресса» (`P_prog`).
