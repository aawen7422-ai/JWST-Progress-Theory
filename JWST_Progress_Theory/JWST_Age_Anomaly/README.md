# Возраст галактики JADES-GS-z14-0 как подтверждение принципа dC/dt = E/ħ: объяснение аномалии JWST

**Статус:** Опубликованный препринт, Февраль 2026
**DOI:** 10.5281/zenodo.18639378
**Часть:** Исследовательская программа «Теория прогресса как квантовая фаза» (Статья №10)

## О чём эта работа

Наблюдения JWST обнаружили галактики, которые «старше Вселенной»:
- **JADES-GS-z14-0** (z = 14.0) — сформировалась через 290 млн лет после Большого взрыва
- **JADES-1050323** (z = 6.9) — её возраст > возраста Вселенной в ΛCDM (4.7σ)

Стандартная модель ΛCDM не может это объяснить. Теория прогресса — да.

## Главный результат

В ранней Вселенной существовала **внутренняя энергия сложности U**, пропорциональная плотности энергии. Для галактик с z > 10:

U ≈ (0.5–1.5) · Mc²

Это даёт **эффективный возраст**:

t_eff = t_ΛCDM · (1 + U/Mc²)

Для JADES-1050323: t_eff / t_ΛCDM ≈ 1.2–1.3 → галактика кажется «старше Вселенной».

## Ключевые выводы

1. ΛCDM неполна — она не учитывает U.
2. Теория прогресса предсказала аномалии JWST в 2025 году.
3. Новые данные (2026) подтверждают предсказания.
4. Все галактики с z > 10 должны иметь избыток сложности.
