# Операционализация параметров сложности C и энергии структуры U

**Статус:** Опубликованный препринт  
**DOI:** 10.5281/zenodo.18682462
**Дата:** Февраль 2026    
**Часть:** Исследовательская программа «Теория прогресса как квантовая фаза» (Статья №11)

## О чём статья
Ранее параметры C (сложность) и U (энергия сложности) использовались как теоретические абстракции.  
Эта статья даёт астрономам **конкретные рецепты**, как вычислить их по реальным наблюдениям.

## Что сделано
- **C_gal** привязана к: массе, цвету (g-r), скорости вращения, возрасту.
- **U** выражена через: sSFR, число спектральных линий, ИК-светимость, градиенты цвета.
- Приведены рабочие формулы и примеры расчётов для:
  * Млечного Пути
  * Солнца
  * пульсара
  * молодой галактики из TNG

## Ключевые формулы
**C_obs:**
```
C_obs = (M_*/10^10)^0.75 × [1+ln(1+t_rot/10^8)] × (1+t_age/10^9)^-0.7 × [1-(g-r)/0.95]
```

**U_obs (для галактик):**
```
U_obs = ħ · (sSFR/10^-8) · (N_lines/100) · (1 - L_IR/L_UV) · (M_*/10^10)
```
