# Термодинамика прогресса: связь dC/dt с производством энтропии и свободной энергией

**Статус:** Опубликованный препринт, [Дата]
**DOI:** [10.5281/zenodo.18623248]
**Часть:** Исследовательская программа «Теория прогресса как квантовая фаза» (Статья №8)

## Основная идея
Разрешается кажущееся противоречие между Вторым началом термодинамики (всё стремится к беспорядку) и ростом сложности (dC/dt > 0) в звёздах, галактиках и живых системах.

## Ключевой результат
Принцип прогресса dC/dt = E/ħ не противоречит термодинамике, а является её следствием для открытых систем:
```
dC/dt = F/ħ + (TS)/ħ
Ξ_thermo = (dC/dt)_pot / (Tσ) >> 1
```
где F — свободная энергия, σ — производство энтропии.

## Что даёт статья
1. **Энергия сложности U** интерпретируется как часть свободной энергии, запасённая в структуре.
2. **Критерий Ξ** получает термодинамическое обоснование.
3. **Парадокс «прогресс vs энтропия»** решён: рост сложности оплачивается производством энтропии в окружающую среду.
4. Расчёты для звезды и ячейки Бенара показывают универсальность механизма.
